# OCR Fonts

Bundled fixed-digit OCR fonts make timer and scoreboard template generation
deterministic across local development, tests, and production.

Included fonts:

- `DejaVuSans-Bold.ttf`
  - Source: DejaVu Fonts 2.37 release archive,
    `dejavu-fonts-ttf-2.37.zip`
  - Upstream: https://github.com/dejavu-fonts/dejavu-fonts/releases/tag/version_2_37
  - License: DejaVu Fonts license, derived from the Bitstream Vera license with
    public-domain additions.
- `Roboto-wdth-wght.ttf`
  - Source: Google Fonts repository, `ofl/roboto/Roboto[wdth,wght].ttf`
  - Upstream: https://github.com/google/fonts/tree/main/ofl/roboto
  - License: SIL Open Font License 1.1.

Only `.ttf` files in this directory are used for fixed-digit OCR templates.
